Apple Health Sync

whoof runs entirely on your phone — these import your Apple Health data with no server and nothing leaving the device.

📄 Import export.xml (full history)

  1. iPhone Health app → profile photo → Export All Health Data.
  2. Unzip the file, then tap 📄 Import and pick export.xml.

📱 One-tap Shortcut (live values)

Install a Shortcut that reads the metrics you want and opens …/#health=<data> back into whoof.
